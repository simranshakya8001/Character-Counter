# Character Counter 🚀

A simple Character Counter built using JavaScript that tracks the number of characters entered by the user in real-time.

## 📌 Features:
- Live character counting as you type
- Shows remaining character limit (20)
- Changes text color when limit exceeds

## 🛠️ Technologies Used:
- HTML
- CSS
- JavaScript

## 📚 What I Learned:
- DOM Manipulation
- Event Handling
- Working with user input

## ▶️ How to Use:
1. Open index.html in your browser
2. Start typing in the input field
3. See the remaining character count update instantly
